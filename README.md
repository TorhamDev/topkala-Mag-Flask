# topkala-Mag-Flask

> topkalamag project with flask and admin panel

## How to run this project?

```bash
git clone https://github.com/TorhamDev/topkala-Mag-Flask.git

cd topkala-Mag-Flask

python -r requirements.txt

cd TorhamBLOG

export FLASK_APP=__init__

python -m flask run

# or

flask run

```
